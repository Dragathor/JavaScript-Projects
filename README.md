### Pizza Menu
A simple menu that is used to order pizza. It has various selactable sizes for the pizzas as well as many toppings. 
It takes the secections from the user and displays the price and selected options at the bottom of the screen. 

### Tic-Tac-Toe Game
The simple game of tic tac toe. Player places a token and the computer then randomly rolls for it's own placement.

### Simple Calculator
The most basic 4 function calculator that has allows the user to add, subtract, multiply and divide with decimal places.
Has a slideshow behind the keys.

### Basic ToDo List
An input window that creates a list of your tasks in below the input.
Tasks are stored in local browser memory for later reference until removed by the user clicking the remove X button.
